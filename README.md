# Personal Portfolio Website

A professional, responsive portfolio website showcasing projects, skills, and contact information.

## Features

- **Home Section**: Eye-catching hero section with introduction and call-to-action
- **Projects & Awards**: Showcase of 4 projects with descriptions and technology tags
- **Skills Section**: Visual representation of technical skills with progress bars
- **Contact Information**: Multiple ways to get in touch (Email, LinkedIn, GitHub, Phone)
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling between sections
- **Professional Design**: Consistent colors, spacing, and typography throughout

## Live Demo

Open `index.html` in your browser to view the portfolio.

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Grid, Flexbox, Custom Properties, Animations)
- Vanilla JavaScript (Intersection Observer, Smooth Scrolling)

## How to Run

1. Clone the repository
2. Open `index.html` in your web browser
3. Or use a local server:
   ```bash
   python3 -m http.server 8080
   ```
   Then navigate to `http://localhost:8080`

## Customization

To personalize the portfolio with your own information:

1. **Edit `index.html`**:
   - Update the home section text
   - Add your own projects and awards
   - Update your skills
   - Change contact information (email, LinkedIn, GitHub, phone)

2. **Modify `styles.css`**:
   - Change colors in the `:root` CSS variables
   - Adjust spacing, fonts, or other styling

3. **Update `script.js`**:
   - Customize animations or interactions as needed
